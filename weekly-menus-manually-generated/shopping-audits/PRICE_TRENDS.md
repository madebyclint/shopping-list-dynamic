## How to View the Trend Charts

Open [price-trends.html](./price-trends.html) in any browser for:
- Bar chart of % price changes (biggest movers)
- Weekly spend vs budget estimate chart
- Filterable table of all tracked items with price history

> The HTML file reads `price-history.json` from the same folder. Open it by running `npx serve .` in the `shopping-audits/` directory, then visiting `http://localhost:3000/price-trends.html`.
> Alternatively, just double-click to open in a browser — it will show a fallback snapshot for the key items.
